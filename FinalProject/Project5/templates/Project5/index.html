About The Project

This is a project for the Utah State University class CS 5890/6890 Practical Natural Language Processing taught by Dr. Qiwei Gan, Ph.D. This is a group assignment that was accomplished by Jayden Bates, Makalee Beelek, KC Gubler, and Braeden Stewart. From the options present, the group decided to choose Project 5: Medical Notes Named Entity Recognition Platform (Application-oriented).

Goal:

Given medical notes, highlight different medical related entities such as diseases and their codes, symptoms, medications, lab tests, etc, display in the webpage.

Skills needed:

  1. CITI training for handling medical sensitive data
  2. Become authorized user of the sensitive medical data
  3. Data preprocessing and manipulation (regular expression, pandas lib)
  4. Named entity recognition
  5. Medical term code resolution
  6. Data visualization
  7. Web development
